# uconf

`uconf` is a SageMath-based Python library for combinatorial operad and
cooperad models used in algebraic topology and configuration-space
computations.

The initial HTML documentation is built directly from the docstrings in
`src/uconf`, with a small amount of handwritten structure around the generated
API pages.
